- 票数要使用同一个静态值

- 为保证不会出现卖出同一个票数，要 java 多线程同步锁

- 创建一个站台类 Station，继承 Thread，重写 run 方法，在 run 方法里面执行售票操作。售票要使用同步锁：即有一个站台卖这张票时，其他站台要等这张票卖完
